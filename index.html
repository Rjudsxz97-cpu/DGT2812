Aplicativo de Avaliação de Destinos Turísticos com Flutter

Objetivo da Prática

O objetivo desta prática foi desenvolver um aplicativo móvel utilizando o framework Flutter para uma agência de viagens fictícia. O aplicativo tem como finalidade apresentar destinos turísticos aos usuários, permitindo a visualização de informações detalhadas sobre cada local, incluindo imagens, descrições, avaliações e funcionalidades interativas como ligação telefônica, visualização de rotas e compartilhamento.

Análise Crítica e Desenvolvimento

2. Implementação da Interface

  • AppBar: Título "Destinos Turísticos" com azul personalizado.
  • ListView: Container scrollável para todo o conteúdo.
  • Row e Column: Organização lógica dos elementos visuais.
  • Icon: Uso de Icons.star para avaliações e ícones funcionais.

3. Seção de Título (titleSection)

Exemplo de implementação do Widget de título:

Widget titleSection = Container(
  padding: const EdgeInsets.all(32),
  child: Row(
    children: [
      Expanded(
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            Text('Praia de Copacabana', style: TextStyle(fontWeight: FontWeight.bold)),
            Text('Rio de Janeiro, Brasil', style: TextStyle(color: Colors.grey[500])),
          ],
        ),
      ),
      Icon(Icons.star, color: Colors.amber),
      const Text('4.8'),
    ],
  ),
);

4. Seção de Botões (buttonSection)

Botão Ícone Função
LIGAR phone Abre o discador com número da agência
ROTA near_me Abre o Google Maps com coordenadas
COMPARTILHAR share Abre menu de compartilhamento nativo

5. Seção de Texto (textSection)

Descrição detalhada com texto justificado e padding de 32 para melhor legibilidade, destacando o calçadão de Copacabana e atividades locais.
